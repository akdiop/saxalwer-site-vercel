# SaxalWér — Site officiel

Plateforme numérique de santé sexuelle et reproductive féminine au Sénégal.

## 🚀 Déploiement Vercel

```bash
npm install
npm run dev      # Développement local
npm run build    # Build production
```

## 🔑 Variables d'environnement Vercel

Déjà configurées sur Vercel :
- `NEXT_PUBLIC_SUPABASE_PROJECT_ID`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

## 📚 Documentation

- `DESIGN_SYSTEM.md` — guide du design system (couleurs, classes, composants)
- `lib/SUPABASE_SCHEMA.md` — schéma SQL à exécuter dans Supabase

## 📁 Structure

```
saxalwer-website/
```
